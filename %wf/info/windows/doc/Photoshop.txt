Выделение + Shift = Добавить к выделению.
Выделение + Alt = Вычесть из него.

D - Цвета чёрный/белый.
X - Обменять цвета.
Alt + Shift + Ctrl + S = Меню сравнительной характеристики сохранения изображения в разных форматах с отображением размера и качества.

Lab  Color  - Интригующая система хранения цвета через яркость и оттенок.
Смешивание слоев - нечто. Ибо позволяет из белого человека сделать негра.
Screen - прозрачность.
Multiplay - автоцветоподстройка под фон! Долой головную боль с освещением
А их стили - позволяют создать капельки воды на листке!
Иконка фотика справа внизу - Quick Mask - надо попробывать ибо это предположительно может позволить градиентный переход!


С - Crop. Через него (плюс флаг перспективы) можно искажать или восстанавливать перспективу изображений.
B - Brush.
Ctrl + T = Масштабирование слоя

Pen +  ctrl = Изменение положения точек

Alt при передвигании ползунка в каждом цвете при коррекции цветопередачи. - до появления пикселей другого цвета - это будут крайние точки гистограммы.

Shift + Пипетка = отметка образца пипетки на рисунке. Маркер.

Наведение резкости через "unsharpen mask" + "luminosity" mix

Alt + Щелчек по слою - выделение загружается из его контрастной области.
